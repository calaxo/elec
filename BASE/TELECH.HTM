Téléchargement du cours

Les divers chapitres de ce cours ont été compressés sous forme d'archives ZIP, format universel Mac, PC, Unix. Chaque archive décompressée se trouvera (normalement) dans un dossier ayant le nom donné dans la fenêtre ci-dessous. Placez tous les dossiers décompressés au même niveau dans un dossier unique (son nom et son emplacement dans le disque n'importent pas). Ne modifiez pas l'arborescence ni les noms des dossiers et fichiers, et tout fonctionnera correctement.

Important !! Les possesseurs de Mac effaceront le suffixe .ZI0 du nom de chaque dossier décompressé, et les possesseurs de PC jetteront à la corbeille le dossier XtraStuf.mac situé à l'intérieur de chaque dossier décompressé.

Le module "BASE" est obligatoire, c'est le moteur de l'ensemble. Pour lancer le cours, double-cliquez le fichier DEPART.HTM situé dans le dossier BASE. Vous pouvez lancer aussi directement un chapitre en double-cliquant le fichier .HTM portant le nom du chapitre et situé dans le dossier relatif à ce chapitre (ex : TRANS.HTM dans le dossier TRANS pour lancer le cours sur le transistor bipolaire)

Ensuite, chaque chapitre peut être téléchargé séparément : même si vous n'avez pas tous les chapitres, l'ensemble fonctionnera quand même. Téléchargez au plus juste ! (France Télécom me hait, et c'est ma joie :-).
