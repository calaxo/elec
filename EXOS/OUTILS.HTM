Outils de calcul

MathPad

Logiciel freeware (gratuit) pour Macintosh : calcul vectoriel

Kit débutants

Deux fichiers MathPad pour démarrer en douceur avec ce logiciel

basic

Bibliothèque de calculs de base utilisée par MathPad : obligatoire

wave

Bibliothèque MathPad servant à générer des signaux (temps/tension), à faire du filtrage élémentaire...

system

Bibliothèque MathPad utilisée pour tous les calculs de systèmes bouclés (tous les montages ci-dessous : obligatoire)

Euler

Logiciel freeware (gratuit) pour PC (versions Linux et windoze disponibles) : calcul vectoriel et matriciel

Décompression de fichiers

MacZip

Logiciel de décompression des fichiers zippés (les fichiers MathPad des exercices sont zippés, donc lisibles par des possesseurs de PC)

Stuffit expander

Logiciel de décompression standard sur Mac : les bibliothèques MathPad sont à ce format. Normalement intégré par tous les navigateurs web pas trop vieux. Mais, en cas de besoin...
