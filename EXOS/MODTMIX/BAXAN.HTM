Correcteur Baxandall


But de l'exercice

On se propose d'étudier un correcteur de type  baxandall  : ce correcteur correspond aux commandes graves/aigus des amplificateurs HIFI, et permet d'atténuer et d'amplifier les graves et les aigus séparément autour d'une fréquence centrale d'environ 1kHz.

 

 

Méthodologie

Nous allons faire cette étude à l'aide de la méthode des systèmes bouclés. Nous allons d'abord identifier les blocs de ce système, puis les réduire à l'aide du théorème de thévenin et du théorème de superposition.

 

A partir des éléments fournis, les personnes désireuses de faire le calcul  à la main  pourront développer les expressions sans problèmes (c'est juste un peu long !). Ici, nous allons utiliser un logiciel de calcul vectoriel (MathPad, pour Macintosh, et Euler pour PC -license GNU).

Application numérique

Voilà ! Pour rentrer tout ceci dans le logiciel, on va procéder rigoureusement comme ci-dessus. A la main, ça sera plus difficile... Il faut en effet déveloper toutes les équations pour ensuite les réduire !

Voilà à priori une excellente raison pour apprendre à se servir d'un logiciel de calcul vectoriel : ça sera à peine plus long !!!

Calcul version Euler (PC Linux et windoze) : téléchargez l'archive (format tar.gz) réalisée par Eric Boucharé. Sous windoze, mettez les programmes dans le dossier "progs" d'Euler, sous Linux, ça n'a pas d'importance, et dans les deux cas, ouvrez le fichier "baxandall.en" depuis Euler.

Calcul version MathPad : téléchargez le programme (visualisez-le dans le navigateur), il est documenté et les variables ont le même nom que celles citées ci-dessus.

Notez qu'avec MathPad vous ne pourrez pas avoir les faisceaux de courbes comme représenté ci-dessous : j'ai fait une autre version du programme, avec des ruses (nécessaires) qui rendent totalement illisible l'ensemble...
