Commande "cross fader"

Schéma

Plusieurs des tables de mixage présentées ici posèdent des voies couplées deux à deux : cette configuration permet d'adjoindre un potentiomètre cross-fader pour passer de l'une à l'autre avec un effet de fondu-enchaîné.

Le module présenté ci-dessous va plus loin : il permet d'intercaler un module cross-fader entre deux sources (entrées ligne seulement, pas des platines phono RIAA, mais ce n'est pas un problème, vu la raréfaction de ces dernières...) et une table de mixage existante sans avoir à casser la dite table pour lui adjoindre cette fonction.
