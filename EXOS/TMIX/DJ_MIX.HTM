Table "disk jockey" 5 voies DJ_MIX

 

Explications générales

On se reportera à ce qui a été dit pour la table MD05S

 

Descriptif

Entrées :

  • une entrée micro
  • deux entrées phono
  • deux entrées ligne


Sorties :

  • vu-mètre 6 leds
  • prise casque avec système de pré-sélection
  • une sortie enregistrement
  • sortie master


Autres :

  • baxandall sur l'entrée micro
  • balance sur l'entrée micro
  • voice over sur le micro
  • quatre entrées phono et lignes couplées deux à deux avec cross fader

Cette table présente une particularité que n'ont pas les autres : une fonction "voice over" qui permet de limiter automatiquement le niveau de la musique (le niveau est réglable) quand on parle dans le micro, ce qui en fait une table de choix pour les soirées animées.

Les entrées phono et ligne sont couplées deux à deux : pour chaque groupe de deux entrées, il faut choisir à l'aide d'un interrupteur soit l'entrée phono, soit l'entrée ligne. Cette "limitation" n'en n'est pas vraiment une, sauf si on veut tout mélanger et faire ainsi une belle cacophonie, mais par contre, elle autorise l'utilisation d'un cross fader, qui ne fonctionne que sur deux voies. En couplant les voies par deux, on peut utiliser le cross fader avec quatre sources.

Le reste de cette table est classique.
