Table 6 voies MM04X

Explications générales

On se reportera à ce qui a été dit pour la table MD05S

 

Descriptif

Entrées :

  • une entrée micro
  • deux entrées phono
  • trois entrées ligne


Sorties :

  • prise casque avec système de pré-sélection
  • deux sorties enregistrement
  • sortie master


Autres :

  • quatre entrées phono et lignes couplées deux à deux
  • troisième sortie ligne indépendante

Cette table est relativement simple : elle est une bonne base pour des gens qui veulent s'initier au mixage. A noter qu'on peut adjoindre un cross fader aux entrées couplées (voir par exemple table M2).
